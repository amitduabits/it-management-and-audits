# Screenshots

This directory is reserved for screenshots of the IT Audit Toolkit in action. Screenshots can be generated by running the toolkit commands and capturing the terminal and report output.

---

## Recommended Screenshots

To build a complete visual reference, capture the following:

### Terminal Output

1. **New Engagement Creation**
   - Command: `python -m src.cli new-audit --name "Demo Audit" --client "Demo Corp" --lead-auditor "Auditor" --output data/demo.json`
   - Shows: Engagement creation confirmation panel, available checklists table

2. **Checklist Listing**
   - Command: `python -m src.cli list-checklists`
   - Shows: Table of all available checklists with control counts and framework references

3. **Checklist Execution**
   - Command: `python -m src.cli run-checklist --audit data/demo.json --checklist access_control`
   - Shows: Interactive control assessment prompts with descriptions and test procedures

4. **Finding Addition**
   - Command: `python -m src.cli add-finding --audit data/demo.json --title "Test Finding" --area "Access Control" --severity high --description "Description"`
   - Shows: Finding creation confirmation with severity badge

5. **Risk Calculation**
   - Command: `python -m src.cli calculate-risk --audit data/sample_audit.json --verbose`
   - Shows: Risk posture panel, compliance table, severity distribution, remediation roadmap

6. **Report Generation**
   - Command: `python -m src.cli generate-report --audit data/sample_audit.json --output reports/demo.html --text-summary`
   - Shows: Report generation confirmation and text executive summary

### HTML Report

7. **Report Header and Executive Summary**
   - Shows: Professional header with engagement metadata, metric cards, severity distribution

8. **Compliance Dashboard**
   - Shows: Per-area compliance bars with color coding

9. **Risk Heat Map**
   - Shows: 5x5 color-coded risk matrix with control counts

10. **Detailed Findings**
    - Shows: Finding cards with severity-colored headers and structured content

11. **Remediation Roadmap**
    - Shows: Numbered priority list with severity badges and timelines

## Generating Screenshots

### Using the Sample Data

```bash
# Generate a complete report from sample data
python -m src.cli calculate-risk --audit data/sample_audit.json
python -m src.cli generate-report --audit data/sample_audit.json --output reports/sample_report.html

# Open the report in a browser for screenshots
# Windows:
start reports/sample_report.html
# macOS:
open reports/sample_report.html
# Linux:
xdg-open reports/sample_report.html
```
