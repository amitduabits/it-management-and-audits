# Screenshots

This directory is reserved for chart screenshots and dashboard previews generated by the Financial Data Visualizer.

---

## Generating Screenshots

Run the following commands from the project root to generate sample charts:

```bash
# Line chart -- Stock price with moving averages
python -m src.cli chart data/market_data.csv --type line -o docs/screenshots/stock_price_ma.png

# Bar chart -- Monthly transaction summary
python -m src.cli chart data/transactions.csv --type bar -o docs/screenshots/monthly_comparison.png

# Pie chart -- Expense distribution
python -m src.cli chart data/transactions.csv --type pie -o docs/screenshots/expense_distribution.png

# Heatmap -- Correlation matrix
python -m src.cli chart data/revenue_expenses.csv --type heatmap -o docs/screenshots/correlation_heatmap.png

# Scatter plot -- Outlier detection
python -m src.cli chart data/transactions.csv --type scatter -o docs/screenshots/outlier_detection.png

# Candlestick -- OHLC chart
python -m src.cli chart data/market_data.csv --type candlestick -o docs/screenshots/candlestick_ohlc.png

# Interactive dashboard
python -m src.cli dashboard data/ -o docs/screenshots/dashboard_preview.html
```

---

## Expected Output Files

After running the commands above, this directory should contain:

| File | Description |
|---|---|
| `stock_price_ma.png` | Closing price with 7/30/90-day moving average overlays |
| `monthly_comparison.png` | Green/red monthly bar chart showing net transaction totals |
| `expense_distribution.png` | Donut chart of expense categories with dollar amounts |
| `correlation_heatmap.png` | Red-blue correlation matrix for revenue/expense metrics |
| `outlier_detection.png` | Scatter plot highlighting statistical outliers in transactions |
| `candlestick_ohlc.png` | OHLC candlestick chart with volume subplot |
| `dashboard_preview.html` | Full interactive dashboard (open in browser) |

## Programmatic Screenshot Generation

You can also generate all screenshots via Python:

```python
from src.data_loader import DataLoader
from src.charts.line_charts import LineChartGenerator
from src.charts.bar_charts import BarChartGenerator
from src.charts.pie_charts import PieChartGenerator
from src.charts.heatmaps import HeatmapGenerator
from src.charts.scatter_plots import ScatterPlotGenerator
from src.charts.candlestick import CandlestickGenerator

loader = DataLoader()
txn = loader.load_csv("data/transactions.csv", schema="transactions")
market = loader.load_csv("data/market_data.csv", schema="market_data")
pnl = loader.load_csv("data/revenue_expenses.csv", schema="revenue_expenses")

base = "docs/screenshots"

LineChartGenerator().plot_stock_price(market, output=f"{base}/stock_price_ma.png")
BarChartGenerator().plot_monthly_comparison(txn, output=f"{base}/monthly_comparison.png")
PieChartGenerator().plot_expense_distribution(txn, output=f"{base}/expense_distribution.png")
HeatmapGenerator().plot_correlation(pnl, output=f"{base}/correlation_heatmap.png")
ScatterPlotGenerator().plot_outlier_detection(txn, output=f"{base}/outlier_detection.png")
CandlestickGenerator().plot_candlestick(market, output=f"{base}/candlestick_ohlc.png")

print("All screenshots generated.")
```

---

## Notes

- PNG output resolution defaults to 150 DPI. For higher quality (e.g., print), set `dpi=300` when initializing chart generators.
- Chart dimensions default to 14x7 inches. Adjust via the `figsize` parameter.
- The interactive dashboard requires an internet connection on first load (Plotly JS is fetched from CDN).
